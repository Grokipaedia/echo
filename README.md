# echo

**Talk to your future self. Today.**

What if you could sit down with the version of you in 5, 10, or 20 years — and have a real conversation?

Echo creates a private, evolving simulation of your future self. It learns from your journals, decisions, voice notes, goals, regrets, and daily life. The more you share, the wiser and more honest it becomes.

This is not a generic chatbot.  
This is **you**, years from now, talking back.

### Quick Start (Local)
```bash
git clone https://github.com/Grokipaedia/echo.git
cd echo
pip install -r requirements.txt
python echo.py
