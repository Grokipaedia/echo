# echo

**Talk to your future self. Today.**

**Latest:** Added **Challenge Mode** (future self can push back) + full profile storage + journal timestamps.  
The more you share, the wiser your future self becomes.
What if you could have a real, ongoing conversation with the version of you in 5, 10, or 20 years?

Echo creates a private, evolving simulation of your future self. It learns from your journals, decisions, voice notes, goals, regrets, and daily life. The more you share, the wiser, more honest, and more personal it becomes.

This is not a generic chatbot.  
This is **you**, years from now, talking back.
